# mob-sOmnistack11
Aplicação dos aplrendizados da Semana Omnistack 11.0


*Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

   - React;
   - Mongodb;
   - Node.js (on backend);


Run the commands on root folder:

   npm i (to push and install all dependencies)


To start project:

   npm start
